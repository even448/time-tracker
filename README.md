# ⏳ 时光清单 & 待办 (Time Tracker & Todo)

一个现代化、高颜值的个人时间管理工具，集成了倒数日记录与待办事项管理功能。支持 PWA，可作为原生应用安装到手机或桌面。

## ✨ 主要功能

### 📅 倒数日 & 纪念日
*   **多种类型支持**：纪念日、考试、生日、工作等。
*   **智能重复**：支持按年、月、周自动重置目标日期（适合生日、发薪日）。
*   **正数日模式**：对于过去的日期（如恋爱纪念日），自动计算“已累计”天数。
*   **自定义背景**：支持上传图片作为卡片背景，自动适配遮罩层。
*   **手势操作**：支持左滑删除卡片。

### ✅ 待办事项 (Todo)
*   **子任务系统**：支持添加检查清单 (Checklist)，完成子任务自动更新主任务进度。
*   **可视化进度**：精美的圆环进度条，直观展示任务完成度。
*   **Git 风格日志**：记录每次进度更新的备注和标签 (Fix, Feat, Docs 等)。
*   **分区管理**：支持自定义分区（如：工作、生活、学习），让任务井井有条。

### 📊 统计与成就
*   **贡献热力图**：模仿 GitHub 的绿色方块热力图，可视化你过去一年的奋斗历程。
*   **数据导入/导出**：支持导出所有数据为 JSON 备份，防止数据丢失。

### 📱 现代化体验 (PWA)
*   **离线访问**：基于 Service Worker 缓存核心资源，无网络也能查看。
*   **安装到主屏幕**：支持添加到手机或电脑桌面，拥有独立图标和启动页，体验接近原生 App。
*   **深色模式**：自动跟随系统或手动切换深色/浅色主题。
*   **通知提醒**：支持浏览器通知，在倒数日或待办截止前发送提醒。

## 🛠️ 技术栈

*   **核心**：原生 HTML5, CSS3, JavaScript (ES6+)
*   **样式**：[Tailwind CSS](https://tailwindcss.com/) (通过 CDN 引入)
*   **图标**：[FontAwesome](https://fontawesome.com/)
*   **图表**：[Chart.js](https://www.chartjs.org/)

## 🚀 快速开始

### 本地运行
1.  克隆或下载本项目。
2.  直接用浏览器打开 `index.html` 即可使用。
3.  为了体验 PWA 功能，建议通过本地服务器运行（如 VS Code 的 Live Server 插件）。

### 部署 (GitHub Pages)
本项目为纯静态应用，非常适合部署到 GitHub Pages：

1.  Fork 本仓库或上传代码到你的 GitHub 仓库。
2.  进入仓库 **Settings** -> **Pages**。
3.  在 **Build and deployment** 下，选择 `main` 分支并保存。
4.  等待部署完成后，访问提供的链接即可。

## 📝 使用说明

1.  **添加倒数日**：点击首页右下角的 `+` 按钮，选择日期、颜色和重复模式。
2.  **添加待办**：切换到底部“待办”标签页，点击 `+` 创建任务。
3.  **数据备份**：点击右上角齿轮图标 ⚙️，选择“导出”即可下载 JSON 数据备份。
